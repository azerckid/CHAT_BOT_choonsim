## 1. API 원가 분석

### 1.1 Gemini 3 Flash 가격 체계 (100만 토큰당)
*   **상태**: 유료 티어 기준 (Paid Tier)
*   **입력(Input)**: $0.50 / 1M tokens
*   **출력(Output)**: $3.00 / 1M tokens

### 1.2 평균 대화 원가 계산
*   **사용자 시나리오**: 1회 채팅 시 평균 1,000 토큰 소모 (입력 800 + 출력 200 가정)
    - 입력 비용: $0.50 * (800 / 1,000,000) = $0.0004
    - 출력 비용: $3.00 * (200 / 1,000,000) = $0.0006
*   **합계 원가**: **$0.001 / 1회 대화** (한화 약 1.4원)

## 3. 최종 정책 제안

### 3.1 환율 고정 (The Golden Ratio)
*   **기준 환율**: **$1 = 1,000 CHOCO**
*   **가치 산정**: **1 CHOCO = $0.001**

### 3.2 기대 효과
1.  **브랜드 인지**: "1,000 CHOCO = 1달러"라는 가장 직관적인 화폐 단위를 구축하여 유저의 가격 이해도를 극대화함.
2.  **미래 확장성**: 그림, 영상 등 고비용 기능 도입 시에도 "영상 1번에 2,000개(2달러)"와 같이 사용자 입장에서 합리적인 숫자로 과금 가능.
3.  **현행 유지**: 현재 판매 중인 Starter Pack($5 / 5,000 CHOCO)과 완벽하게 일치하여 별도의 패키지 조정이 필요 없음.
