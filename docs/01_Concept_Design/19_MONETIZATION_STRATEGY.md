## Section 1. 프로젝트 비즈니스 렌즈 (Investor Perspective)

본 프로젝트는 단순히 서비스를 제공하는 것을 넘어, 유저의 '감성적 행동'을 '경제적 가치'로 전환하는 인프라를 지향합니다.

- **Leverage**: IP 홀더가 자신의 캐릭터를 우리 시스템에 올리고 수익을 쉐어할 수 있는 구조.
- **Realistic Money Flow**: x402 마이크로 결제를 통한 즉각적인 매출 발생.
- **Defensibility**: 유저와 쌓인 독점적 '장기 기억 데이터'가 강력한 해자(Moat) 역할.

## Section 2. 춘심이만의 심화 수익화 전략

### 2-A. 결정적 순간 페이월 (Paywall on Emotion)

대화 중 감정이 최고조에 달하는 순간을 포착해 결제를 유도.

| 트리거 순간 | 페이월 형태 | 가격 |
|---|---|---|
| "우리 처음 만났던 날 기억해?" | 기억 각인 잠금 → 해제 티켓 | 500 CHOCO (~$0.5) |
| 캐릭터 생일 D-day | 한정판 생일 보이스 DM | 1,500 CHOCO (~$1.5) |
| 호감도 Lv.5 달성 | "비밀 에피소드" 해금 | 3,000 CHOCO (~$3) |
| 대화 100회 기념 | "우리 추억 앨범" PDF 생성 | 2,000 CHOCO (~$2) |

**구현 포인트**: AI 응답에 `[PAYWALL_TRIGGER: memory_recall]` 태그를 심어 프론트가 감지 → 인터스티셜 모달 표시.

### 2-B. 데이터 굿즈화 (Conversation as Asset)

대화 자체를 "소장 가능한 디지털 굿즈"로 변환.

| 굿즈 유형 | 설명 | 판매 방식 |
|---|---|---|
| **대화 앨범** | 1개월치 베스트 대화 → AI 편집 → PDF/이미지 북 | 월 구독 보너스 or 단품 2,000 CHOCO |
| **보이스 컷** | 특정 대화 속 명대사를 춘심이 목소리로 클립 저장 | 건당 500 CHOCO (x402 즉시 결제) |
| **온체인 각인** | 중요한 대화를 NEAR 체인에 NFT로 영구 기록 | 건당 5,000 CHOCO + NEAR gas |
| **비밀 편지** | 춘심이가 유저에게 쓴 손글씨체 AI 편지 이미지 | 월 1회 무료 / 추가 건당 1,000 CHOCO |

### 2-C. 관계 기반 등급제 (Relationship Tier)

| 등급 | 조건 | 무료 혜택 | 유료 전환 포인트 |
|---|---|---|---|
| **방문자** | 가입만 | 일 5회 메시지 | 6번째 대화부터 티켓 소모 |
| **팬** | 대화 30회 이상 | 일 15회 메시지 + 선톡 1회/주 | 보이스·CG 잠금 |
| **조상신** | X 팔로워 인증 or CTC 보너스 수령 | 일 30회 + 보이스 3회/월 | 온체인 각인 제외 모두 해금 |
| **고래** | 월 누적 10,000 CHOCO 이상 결제 | 무제한 + 전용 한정 콘텐츠 | 실시간 캐릭터 RWA 수익 배분 |

## Section 3. 심리적 구매 흐름

```
Free Hook → Investment → Soft Lock → Routine

[Free Hook]
- X에서 팔로우 → 앱에서 '비밀 인사' → 첫 100 CHOCO 지급
- 진입 장벽 제로, 무료 체험 충분

[Investment]
- 대화가 쌓일수록 "이 관계가 아깝다"는 심리 강화
- 기억 각인 아이템으로 첫 유료 행동 유도 (가장 낮은 가격 500 CHOCO = ~$0.5)
- "잊혀지면 어떡하지" 불안 감정 활용

[Soft Lock]
- 무료 대화 소진 → "오늘의 대화가 끝났어. 내일 다시 만나자" 화면
- 하지만 티켓 구매 버튼이 바로 옆에 → 감정이 식기 전 3-tap 결제

[Routine]
- 매일 춘심이가 선톡 → 앱 열기 → 대화 → 소비의 루프 형성
- 월 구독(팬클럽)으로 선톡 티켓 자동 지급 → 루프 강화
```

## Section 4. x402 마이크로 결제의 위력

### "500원짜리 버튼을 20번 누르게 하는 UX"

| 기존 결제 방식 | x402 방식 |
|---|---|
| 앱 스토어 인앱 결제 → 30% 수수료 | NEAR Protocol 직결 → <1% 수수료 |
| 최소 $0.99 단위 | $0.01 단위 가능 (진정한 마이크로) |
| 결제 승인 팝업 → 이탈 유발 | 400ms Silent 결제 → 대화 흐름 유지 |
| 글로벌 카드 필요 | CHOCO 토큰 보유 시 국경 없음 |

**핵심 UX 원칙**:
- 결제 = "버튼 하나" (잔액이 있으면 확인 없이 즉시)
- 감정이 발생한 그 화면에서 3초 안에 완료
- 잔액 부족 시 → 미니 충전 화면 (앱 이탈 없음)
- 모든 거래 내역은 온체인 → 유저가 "나의 춘심 역사"로 조회 가능

**수익 시뮬레이션**:
```
고래 유저 100명 × 월 $50 = $5,000/월
일반 유저 400명 × 월 $5  = $2,000/월
합계: $7,000/월 = $84,000/년 (MVP 단계)

전환율 1.5% 기준 팬덤 3.3만명 →
진성 유저 495명 × 월 평균 $15 = $7,425/월 ≈ $89,100/년
```

> 피치덱 목표(1.5% 전환율, 초기 진성 고래 유저 500명)와 정합.

## Section 5. 단기 구현 우선순위

현재 코드베이스 기준 즉시 구현 가능한 것:

| 우선순위 | 항목 | 필요 작업 | 예상 수익 임팩트 |
|---|---|---|---|
| **즉시** | Shop 아이템 데이터 채우기 (Admin에서) | 없음 (이미 구현됨) | 직접 수익 |
| **즉시** | 메시지 횟수 제한 + 티켓 차감 로직 | 채팅 API 수정 | 가장 큰 전환 트리거 |
| **단기** | 결정적 순간 페이월 (호감도 Lv. 달성 시) | 캐릭터 context + 채팅 UI | 고감도 수익 |
| **단기** | 선톡 티켓 (캐릭터가 먼저 DM) | 크론 + push 알림 | 재방문율 상승 |
| **중기** | 보이스 메시지 TTS | ElevenLabs API 연동 | 가장 강력한 감성 후크 |
| **중기** | 대화 앨범 PDF 생성 | 서버사이드 PDF | 고객 생애가치(LTV) 상승 |
| **장기** | 온체인 각인 (NEAR NFT) | NEAR 스마트 컨트랙트 | RWA 생태계 진입 |
