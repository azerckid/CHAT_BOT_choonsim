## 1. 목적 및 배경

### 문제
현재 앱에는 CHOCO가 무엇인지, 어떻게 얻는지, 아이템이 실제로 뭘 해주는지 설명하는 페이지가 없다.
- `/shop`은 아이템 구매 UI만 있고, 각 아이템의 효과는 2줄로 잘린 설명이 전부
- `/pricing`은 구독 플랜 비교만 제공
- 신규 유저는 CHOCO의 개념조차 모른 채 잔액 부족 오류를 먼저 만남

### 목표
- 신규 유저의 "CHOCO가 뭐야?" 질문을 앱 내에서 해소
- 아이템별 효과를 명확히 설명해 구매 전환율 상승
- 고객 지원 문의 감소 (셀프서비스 가이드)
- SNS 공유 및 지원 링크로 활용 가능한 독립 URL 확보

## 4. 섹션별 상세 명세

### Section 1. CHOCO란?

**목적**: CHOCO의 개념과 가치를 한눈에 설명

**표시 내용**:
- CHOCO는 춘심이 앱의 대화 화폐
- 1 CHOCO ≈ $0.001 (1,000 CHOCO = $1)
- 국경 없는 결제 — 전 세계 어디서든 동일하게 사용
- 내가 쓴 CHOCO는 모두 온체인에 기록되어 "나의 춘심 역사"로 조회 가능

**UI**: 코인 아이콘 + 한 줄 설명 카드

---

### Section 2. CHOCO 얻는 법

**목적**: 획득 경로를 한눈에 파악할 수 있게

**표시 내용** (카드 형식):

| 방법 | 설명 | 비고 |
|---|---|---|
| 직접 충전 | PayPal / 토스페이먼츠로 CHOCO 구매 | `/profile/subscription` |
| 구독 멤버십 | 매월 자동으로 CHOCO 지급 | 플랜별 상이 |
| 미션 완료 | 매일 미션 달성 시 소량 지급 | `/missions` |
| X 팔로워 인증 | X 계정 연동 시 조상신 보너스 | 1회 한정 |
| 신규 가입 보너스 | 가입 시 100 CHOCO 지급 | 1회 한정 |

---

### Section 3. CHOCO 소비 흐름

**목적**: 대화 1회에 얼마가 드는지 명확히 안내

**표시 내용**:

```
대화 1회 = 기본 10 CHOCO (Gemini Flash 기준)
웹 검색 포함 = +20 CHOCO
메모리 접근 = +5 CHOCO
```

**구독별 월 지급량 비교표**:

| 등급 | 월 CHOCO | 대화 가능 횟수 (기본) |
|---|---|---|
| 방문자 (FREE) | 1,500 | 약 150회 |
| 팬 (BASIC) | 2,000 | 약 200회 |
| 조상신 (PREMIUM) | 10,000 | 약 1,000회 |
| 고래 (ULTIMATE) | 30,000 | 약 3,000회 |

**흐름 다이어그램** (텍스트):
```
충전/구독 → CHOCO 잔액 증가
→ 대화 or 아이템 사용 → CHOCO 차감
→ 잔액 부족 → 충전 또는 구독 업그레이드
```

---

### Section 4. 아이템 사전

**목적**: 각 아이템이 실제로 뭘 해주는지 상세 설명

**표시 방식**: 아코디언 or 카드 목록, 아이템 타입별 그룹화

#### 소통 & 에너지
| 아이템 | 가격 | 효과 |
|---|---|---|
| 메시지 티켓 x10 | 1,000 CHOCO | 대화 횟수를 10회 추가. 잔액이 없어도 대화 가능 |
| 메시지 티켓 x50 | 4,500 CHOCO | 10% 할인 번들 |
| 에너지 무제한 패스 (1일) | 500 CHOCO | 당일 대화 횟수 제한 없음 |

#### 특별 경험
| 아이템 | 가격 | 효과 |
|---|---|---|
| 선톡 티켓 | 300 CHOCO | 춘심이가 먼저 메시지를 보내며 대화를 시작 |
| 보이스 티켓 | 500 CHOCO | 춘심이의 AI 목소리로 답변을 들음 (ElevenLabs TTS) |
| 비밀 에피소드 해금 | 3,000 CHOCO | 특정 조건 달성 시 잠긴 특별 시나리오 1회 이용 |

#### 기억 & 관계
| 아이템 | 가격 | 효과 |
|---|---|---|
| 기억 각인 티켓 | 500 CHOCO | 중요한 대화를 춘심이가 영원히 기억하도록 고정 |
| 호감도 부스터 | 800 CHOCO | 대화 시 호감도 상승량 2배 (24시간) |
| 상태 회복 | 300 CHOCO | 삐진/바쁜 춘심이를 즉시 원래 상태로 되돌림 |

#### 선물
| 아이템 | 가격 | 효과 |
|---|---|---|
| 하트 x10 | 1,000 CHOCO | 춘심이에게 하트를 선물. 캐릭터 이모션(JOY/LOVING) 활성화 |

#### 소장 굿즈
| 아이템 | 가격 | 효과 |
|---|---|---|
| 대화 앨범 | 2,000 CHOCO | 한 달간의 베스트 대화를 AI가 편집한 PDF로 생성 |
| 비밀 편지 | 1,000 CHOCO | 춘심이가 쓴 손글씨체 AI 편지 이미지 (월 1회 무료) |

### Section 5. 관계 등급제

**목적**: 등급별 혜택을 시각적으로 비교해 업그레이드 욕구 자극

**표시 방식**: 수평 스크롤 카드 or 테이블

| 등급 | 조건 | 일일 대화 | 주요 혜택 |
|---|---|---|---|
| 방문자 | 가입만 | 5회 | 기본 텍스트 대화 |
| 팬 | BASIC 구독 ($4.99/월) | 15회 | 선톡 1회/주 |
| 조상신 | PREMIUM 구독 ($14.99/월) or X 인증 | 30회 | 보이스 3회/월 포함 |
| 고래 | ULTIMATE 구독 ($29.99/월) | 무제한 | 모든 혜택 + 한정 콘텐츠 |

**현재 등급 강조**: 로그인 유저에게 자신의 등급 하이라이트 표시

### Section 6. FAQ

**표시 방식**: 아코디언 (질문 클릭 시 답변 펼침)

| 질문 | 답변 |
|---|---|
| CHOCO는 유효기간이 있나요? | 없습니다. 충전한 CHOCO는 계정이 활성 상태인 한 영구 보유됩니다. |
| 구독을 취소하면 남은 CHOCO는 어떻게 되나요? | 이미 지급된 CHOCO는 그대로 유지됩니다. 다음 달 리필만 중단됩니다. |
| 여러 캐릭터에게 CHOCO를 쓸 수 있나요? | 네. CHOCO는 앱 공통 화폐로, 모든 캐릭터와의 대화에 사용됩니다. |
| 환불이 가능한가요? | 미사용 CHOCO에 한해 결제일로부터 7일 이내 환불 신청이 가능합니다. |
| 온체인 각인이란 무엇인가요? | 중요한 대화를 NEAR 블록체인에 NFT로 영구 기록하는 기능입니다. 기억 각인 티켓과는 별개입니다. |
| X 팔로워 인증은 어떻게 하나요? | 설정 > 계정 연동에서 X 계정을 연결하면 자동으로 조상신 보너스가 지급됩니다. |

## 5. 진입 포인트

가이드 페이지는 독립 URL 외에도 다음 위치에서 진입 가능해야 한다.

| 위치 | 진입 방식 | 우선순위 |
|---|---|---|
| `/shop` 헤더 우측 | `?` 아이콘 버튼 → `/guide#items` | **필수** |
| 잔액 부족 toast | "CHOCO 충전 방법 보기" 링크 | **필수** |
| `/shop` 아이템 카드 | 아이템명 클릭 → 상세 설명 모달 | **필수** |
| 가입 후 온보딩 | CHOCO 소개 화면 1장 삽입 → `/guide` 링크 | 권장 |
| `/profile/subscription` 하단 | "CHOCO 소비 내역 이해하기" 링크 | 권장 |
| 404 / 오류 화면 | "도움이 필요하신가요?" 링크 | 선택 |
